## Lock Rejection Evidence

the first World Cup 2026 kickoff is `2026-06-11T19:00:00.000Z`.

Default recorded in `dev/open-questions.md`: outside production only, the server
accepts `x-worldcup-now` so tests can verify closed-match behavior before the
tournament starts. Production ignores the override and uses server time.

`npm run test:v0.2` verifies:

- Match 1 is open at `2026-06-11T18:59:59.000Z`

- Match 2 remains open at `2026-06-11T20:00:00.000Z`
- bracket writes reject at the first tournament kickoff
- group picks reject unknown and knockout match IDs
