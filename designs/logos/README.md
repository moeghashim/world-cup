# WinWorldCup2026 Logo Concepts

These are independent logo explorations for `winworldcup2026.com`.

## Variations

- `winworldcup2026-logo-a-orbit-cup.svg`: circular football orbit with a generic winner cup.
- `winworldcup2026-logo-b-motion-ball.svg`: fast prediction/matchday motion mark with a football and winner spark.
- `winworldcup2026-logo-c-shield-globe.svg`: shield and globe-grid mark for a more structured contest identity.
- `logo-concept-board.png`: generated raster concept board used to explore the three directions.

## Usage Notes

- These are not official FIFA, World Cup, federation, sponsor, player, or trophy marks.
- SVG files are the usable source assets because the text is deterministic and editable.
- The PNG board is for visual direction only.
